# Getting Started

This is where you start. One guide, one outcome: mzspec installed and ready to run.

| Guide | What it covers |
|-------|---------------|
| [Install mzspec](01-install.md) | Prerequisites, one-liner, what gets installed, next steps |

After installing, run `./mzspec docs` in your terminal for a visual workflow overview,
then start your first change with `./mzspec propose "what to build"`.
